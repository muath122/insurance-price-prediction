# Medical Insurance Charges Prediction

[IN PROGRESS]

## Project Overview
This project analyzes a health insurance dataset and builds a machine learning model to predict medical insurance charges based on demographic and health-related features.

## Dataset Features
- age
- sex
- bmi
- children
- smoker
- region

**Target variable:**
- charges

## Objective
The main goal of this project is to predict individual medical insurance charges using machine learning techniques.

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Load and explore the dataset
2. Clean and preprocess the data
3. Encode categorical variables
4. Split the data into training and testing sets
5. Train machine learning models
6. Evaluate model performance
7. Predict insurance charges

## Status
This project is currently in progress.
